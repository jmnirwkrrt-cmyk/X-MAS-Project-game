# USER GUIDE FOR THE GAME IN RUSSIAN LANGUAGE
## Save system
The game supports autosaves and manual saves. Saves are written to AppData/Local/X_MAS_Project
If you accidentally select a new game, the old save is not deleted, but you won’t be able to simply load it. You need to rename it or replace it with a new save. 
Saves are not encrypted and can be edited, but it is recommended to make a backup copy before doing so. 

## Gameplay
On the world map, the gameplay resembles a simplified X-COM UFO DEFENSE, in air combat it resembles a reimagining of Xenonauts, and in ground combat it resembles a simplified Sonic Battle.
To complete the game find a way to stop dr. Eggman and save christmas.

## Management
Air combat:
ESC - restart
Space - pause
Mouse to the edge of the screen or arrows - move the screen

Ground combat:
ESC - menu
F - hit (a series of three hits gives a combo)
G-block
CTRL - superpower 1 or shooting
ALT - superpower 2
Spacebar - jump, two jumps = flight (not for everyone)
Q - toss 
Shift - dash
F1 - first aid kits
F2 - grenades (can only be thrown by snow conscripts).
Arrows - camera control
